# CAN_Bootloader
This Repo includes CAN Bootloader App For STM32F446RE and STM32F103C8T6, the App is Display an message on LCD. 

# SWC : 
1- RCC driver.

2- GPIO driver. 

3- UART driver.

4- CAN driver.

5- SYSTICK driver. 

6- Parser

7- Flash driver.

8- LCD driver.

implemented for both NucleoF446RE and STM32F103 Blue Pill.
